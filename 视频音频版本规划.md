# 视频/音频版本规划

> 为「人机共智：AI时代的协作思维与实践」课程规划多媒体形式

---

## 背景

文字版本已完整建立，考虑扩展为视频/音频形式，以覆盖更多学习场景：
- 满足不同学习风格（视觉型、听觉型）
- 方便通勤/运动时收听
- 提供更直观的内容演示
- 降低学习门槛

---

## 一、视频版本规划

### 1. 形式选择

| 形式 | 时长 | 适用内容 | 制作难度 |
|------|------|----------|----------|
| 动画讲解 | 5-10分钟/章 | 概念解释、流程图 | 中 |
| PPT录屏+配音 | 10-15分钟/章 | 知识串讲 | 低 |
| 真人出镜 | 10-20分钟/章 | 案例分析、互动 | 高 |
| 微课短视频 | 1-3分钟 | 核心概念、要点 | 中 |

### 2. 建议结构（动画讲解类）

```
每个模块 → 4-6集

第1集：章节概述（3分钟）
- 本章学习目标
- 核心概念预告
- 为什么重要

第2-4集：核心内容（每集5-8分钟）
- 细分知识点
- 配合动画/图示
- 案例讲解

第5集：实践指导（5-10分钟）
- 如何应用
- 操作演示
- 常见误区

第6集：思考与练习（3分钟）
- 回顾要点
- 练习题提示
- 下章预告
```

### 3. 内容适配要点

```
需要从文字转化的内容：

✅ 需要增强的部分：
• 流程图→动态演示动画
• 表格→对比图表动画
• 案例→情景再现视频
• 操作步骤→屏幕录制+操作演示

✅ 音频需要额外准备的：
• 背景音乐选择（轻学术风格）
• 音效（重点强调）
• 转场过渡音效

✅ 需要重新设计的部分：
• 互动问答（文字→视频提问+暂停）
• 小组讨论→视频内引导语
```

### 4. 技术栈建议

```
视频制作：
• 动画：Canva、剪映、PowerPoint动画
• 录屏：OBS Studio、Camtasia
• 剪辑：剪映、Premiere Pro、DaVinci Resolve
• 配音：Adobe Podcast（AI降噪）、Audacity

发布平台：
• B站（主平台）
• YouTube
• 小红书（短视频版）
• 微信视频号

格式建议：
• 主视频：1920x1080, H.264, 30fps
• 短视频：9:16竖版 or 1:1方形
• 配套字幕：SRT格式
```

## 二、音频版本规划

### 1. 播客形式

```
播客系列：「人机共智：AI时代的协作思维与实践」电台

形式：对话体（主播+嘉宾）
时长：20-30分钟/集
更新：每周1集

适合内容：
• 章节串讲
• 案例深度讨论
• 专题延伸

设备建议：
• 麦克风：Blue Yeti / 797 Audio
• 录音：Audacity / Adobe Audition
• 降噪：Adobe Podcast / RTX
```

### 2. 有声书形式

```
形式：单人口述
时长：全文朗读
时长：约6-8小时（全书）

适合场景：
• 通勤收听
• 睡前聆听
• 视障人士

制作：
• 文字转语音（AI配音）：ElevenLabs、微软TTS
• 或真人朗读：更高质量但成本高

建议：
• 重点章节真人朗读
• 练习/案例部分AI配音
• 混合制作
```

### 3. 音频要点提炼

```
每个章节制作：
• 完整版：全文朗读（20-40分钟）
• 精华版：核心要点提炼（5-10分钟）
• 问答版：常见问题解答（10-15分钟）

额外制作：
• 思维导图音频版：关键概念串联
• 练习引导音频：冥想/引导式练习
```

---

## 三、制作优先级

### 第一阶段：低成本快速上线

```
优先级1（最容易）：
□ 为每个模块录制「精华版」音频（5分钟）
  → 使用TTS工具快速生成
  → 发布到喜马拉雅/小宇宙

优先级2（容易）：
□ PPT录屏+配音视频（10分钟/章）
  → 使用剪映自动字幕
  → 发布到B站

优先级3（中等）：
□ 配套图文发布（小红书/公众号）
  → 每章3-5张思维导图
  → 核心概念卡片
```

### 第二阶段：高质量内容

```
优先级4（需要投入）：
□ 动画讲解视频（每个模块2-3集）
  → 可以用Canva动画
  → 或外包制作

优先级5（需要团队）：
□ 播客节目录制
  → 需要固定嘉宾
  → 定期录制

优先级6（长期）：
□ 真人出镜课程
  → 需要专业拍摄
  → 完整体系
```

## 六、成本估算

### 自制（低成本）

```
设备：
• 手机/电脑自带（0元）
• 领夹麦：200-500元
• 预算：500元以内

软件：
• 剪映Pro：免费/会员约300元/年
• Canva：免费/Pro约1200元/年
• 配音AI：按字数计费，约0.1元/字
• 预算：约2000元/年

人力：
• 自己制作：0元
• 时间投入：约10-20小时/章
```

### 外包（高质量）

```
动画制作：
• 每集5-10分钟：3000-8000元
• 全套（约30集）：90000-240000元

真人拍摄：
• 每集20分钟：5000-15000元
• 全套：150000-450000元

播客外包：
• 每期30分钟：1000-3000元
• 季度：12000-36000元

建议：
初期自己制作 → 验证内容 → 有资源后外包
```
