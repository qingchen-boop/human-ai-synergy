# 人机协作学 Human-AI Synergy Studies (HASS)

> *让人类在AI时代不只是工具的使用者，而是与AI共同进化的高效协作者。*

---

## 📚 课程概述

人机协作学是一门面向AI时代的整合性跨学科课程，涵盖：

| 模块 | 主题 | 核心问题 |
|------|------|----------|
| 模块一 | AI能力与局限 | AI能做什么？不能做什么？为什么？ |
| 模块二 | 人机协作策略 | 怎样和AI配合达到1+1>2？ |
| 模块三 | AI批判与伦理 | AI对社会的深层影响是什么？ |
| 模块四 | AI时代个人发展 | 个人怎么用AI实现认知增强？ |
| 模块五 | 前沿与未来 | AI将走向何方？我们如何准备？ |

---

## 🎯 你能学到什么

- ✅ 理解大模型的工作原理和真实能力
- ✅ 掌握高效的人机协作策略和提示工程
- ✅ 培养批判性使用AI的思维习惯
- ✅ 设计适合自己的人机协作工作流
- ✅ 理解AI对就业、社会和知识生产的影响
- ✅ 形成面向未来的人机关系认知框架

## 🚀 快速开始

### 第一步：阅读学科纲要
从 `1-学科纲要.md` 开始，了解整体框架。

### 第二步：按模块学习
按顺序学习五个模块，每个模块约4-6学时。

### 第三步：完成实践练习
每个模块都有对应的练习，实践比理论更重要。

### 第四步：设计你的工作流
用课程中学到的方法，设计一套你自己的人机协作流程。

## 📝 课程特色

- **反废话** — 不讲"AI将改变一切"这种空洞话，直奔实操
- **反神话** — 不神化AI，也不危言耸听，理性认知
- **反被动** — 不是教你"怎么用ChatGPT"，是教你"怎么和AI协作"
- **有框架** — 不是碎片化技巧，是系统性思维

---

## 💡 学习建议

1. **边学边用**：每个概念都用你日常的任务去测试
2. **记录反思**：写学习日志，记录你和AI协作的体验
3. **批判思维**：课程内容也是你批判的对象
4. **持续迭代**：AI在变，你的认知也要跟着变
