## 关于课程本身

### Q1：这个课程和"学Python/学AI编程"有什么区别？

| 学Python/AI编程 | 人机协作学 |
|----------------|-----------|
| 学会使用某一种工具 | 学会理解AI的思维模式 |

**简单说：** 编程课教你写代码，这个课教你和AI一起思考。

## 关于AI本身

### Q5：AI会取代人类工作吗？

**会，也不会。**

- AI会取代**部分**工作，特别是重复性高、规则清晰的任务
- 但完全取代人类的工作是少数
- 更可能的是：**工作形态改变**，人类需要适应与AI协作

关键不是"AI会不会取代我"，而是"会用AI的人会不会取代不会用AI的人"。

---

### Q6：AI的回答经常是错的，我该怎么相信它？

**保持批判性信任：**

1. AI说的事实，一定要核实（特别是具体人名、日期、数字）
2. 越是你不熟悉的领域，越要小心
3. AI的语气不代表准确性——它自信地说，可能只是因为统计上这个回答最"像真的"
4. 重要决策永远不要只依赖AI

**正确心态：** AI是一个超强工具，但你是判断者。

---

### Q7：我该用哪个AI？

**没有标准答案。以下是直觉：**

| 需求 | 推荐 |
|------|------|
| 日常对话、学习辅助 | ChatGPT / Claude / Gemini |
| 实时信息查询 | Perplexity / Gemini |
| 长文本分析 | Claude |
| 代码相关 | Claude / GitHub Copilot |
| 图像生成 | Midjourney / DALL-E |
| 中文场景 | Kimi / 文心一言 / 智谱 |

**建议：** 都试试，用你喜欢的方式。没有完美的AI，只有适合你的AI。

---

### Q8：AI会偷我的数据吗？

**需要具体情况具体分析：**

- 大多数AI在**你输入问题时**会使用你的数据（查看他们的隐私政策）
- 有的会用于模型训练，有的不会
- 商业敏感/隐私内容，谨慎输入

**建议：**
- 不要输入你不愿意公开的内容
- 使用隐私模式/企业版（如果有）
- 了解你使用产品的数据政策

## 关于学习方法

### Q9：我尝试用AI帮助学习，但感觉没学到东西？

**可能你用错了方法。**

| ❌ 错误方式 | ✅ 正确方式 |
|------------|------------|
| 让AI替你阅读 | 用AI帮你解析你读不懂的部分 |
| 让AI总结给你 | AI总结后你自己再理解一遍 |
| 不懂就问AI | 先思考，再用AI提问验证 |
| 照搬AI的答案 | 把AI的答案用自己的话重述 |

**核心原则：** AI是放大镜，放大你的思考；不是替代品，替代你的思考。

---

### Q10：我对AI输出很依赖，怎么办？

**警惕信号：**
- 没有AI就无法开始任务
- 分不清哪些想法是你的，哪些是AI的
- 不和AI对话就感到焦虑

**解决方法：**
1. 刻意练习"先自己想，再问AI"
2. 把AI当作"讨论对象"，不是"权威答案"
3. 保留"不用AI"的学习体验
4. 问自己：这件事我有没有AI也想要？

---

### Q11：我觉得AI很强大，不需要学习了？

**这是一个危险的误解。**

- AI强大不代表你不需要学习
- 会用方向盘不代表不需要知道目的地
- 能查地图不代表不需要知道自己在哪

**学习的意义不只是"获取信息"，而是：**
- 建立判断力（知道什么是对的）
- 培养批判性（知道什么时候AI错了）
- 保持主动性（不是被AI推着走）

## 关于伦理和批判

### Q15：我用AI帮我写作业，算作弊吗？

**取决于你怎么用。**

| ✅ 可以接受 | ❌ 不建议 |
|------------|----------|
| 用AI帮你理解题目 | 让AI直接写答案提交 |
| 用AI帮你检查语法 | 把AI写的冒充自己的 |
| 用AI提供思路，你自己写 | 完全不动脑地照搬 |
| 用AI润色你自己的文字 | 让AI代替你的思考过程 |

**核心原则：** 你最终要为你提交的内容负责。AI可以帮你，但学习是你的。

---

### Q16：AI有偏见，我该怎么识别？

**常见偏见类型：**
- 性别偏见（"程序员应该是男性"）
- 种族偏见（某些群体描述的系统性偏差）
- 文化偏见（以西方为中心的视角）
- 时代偏见（对某些事情的理所当然）

**识别方法：**
1. 问"这个回答对所有群体都公平吗？"
2. 尝试不同的问法，看回答是否有差异
3. 用自己的背景去检验
4. 寻找其他来源对比

---

### Q17：AI会不会让人类变笨？

**有可能，如果你被动使用。**

但这不是AI的问题，是使用方式的问题。

**会变笨：** 什么都问AI，不自己思考  
**不会变笨：** 用AI扩展能力，同时保持自己的思考强度

AI是工具，工具会放大使用者的特点。你本来爱思考，用AI会更强大；你本来不爱思考，用AI可能会更懒。
