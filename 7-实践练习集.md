# 人机共智：AI时代的协作思维与实践 - 实践练习集

---

## 练习说明

每个练习都设计为**30-60分钟**，涵盖"知道"到"做到"的转化。

---

## 模块一练习：AI能力与局限

【开放练习，无标准答案】

### 练习1.1：AI能力边界测试（45分钟）

**目标：** 建立对AI能力的直观认知

**任务：**
1. 选择一个你熟悉的领域
2. 列出10个任务，按"AI适合度"排序
3. 实际让AI完成这10个任务
4. 记录AI表现，对比你的预期
5. 总结：AI实际能力 vs 你的预期

**记录模板：**
```
任务 | 我的预期 | AI实际表现 | 差距分析
-----|----------|------------|----------
1.    |          |            |
2.    |          |            |
...
```

---

### 练习1.2：幻觉识别训练（30分钟）

**目标：** 学会识别AI的可能错误

**任务：**
1. 让AI介绍一个你专业领域的具体概念/人物/事件
2. 故意问一些边界问题（比如：非常细节的事实）
3. 观察AI在不确定时如何回应
4. 用其他来源核实AI的回答
5. 记录：哪些地方AI可能"幻觉"了？

---

### 练习1.3：任务分类实践（30分钟）

**目标：** 学会判断什么任务适合AI

**场景：** 你今天有这些事情需要处理：

```
a) 写一封正式的工作邮件
b) 决定下周是否要出差
c) 查找2024年诺贝尔经济学奖得主
d) 给焦虑的朋友一些安慰建议
e) 帮你规划下个月的日程
f) 帮你写一首生日祝福诗
g) 分析你公司的季度报表
h) 决定买哪款手机
```

**任务：**
1. 对每个任务，判断：AI主导 / AI辅助 / 人类主导
2. 说明理由
3. 和同伴讨论，比较判断差异

## 模块二练习：人机协作策略

【开放练习，无标准答案】

### 练习2.1：提示优化（45分钟）

**目标：** 学会写出有效的提示

**任务：**
选择一个你常用的AI，写3个"不好"的提示，然后优化它们。

**示例：**
```
❌ 不好提示1："帮我看看这篇报告"
→ ✅ 优化后："请指出这篇报告的三个主要论点，
    并评价每个论点的论据是否充分"

❌ 不好提示2："给我一些创意"
→ ✅ 优化后："我需要为一个老年健身App想5个推广创意，
    目标用户是60-70岁刚退休的老人，
    偏好积极、有尊严的调性"
```

**你的任务：**
```
❌ 不好提示1：_____________________
→ ✅ 优化后：_____________________

❌ 不好提示2：_____________________
→ ✅ 优化后：_____________________

❌ 不好提示3：_____________________
→ ✅ 优化后：_____________________
```

---

### 练习2.2：任务分解实践（60分钟）

**目标：** 学会设计人机协作流程

**任务：** 设计一个用AI帮助写毕业论文的工作流

```
步骤1：列出论文写作的主要环节
步骤2：标注每个环节的AI参与程度
步骤3：设计人机回环的具体方式
步骤4：确定你作为人类必须把控的环节
步骤5：画出你的"人机协作写作流程图"
```

---

### 练习2.3：多AI对比（45分钟）

**目标：** 体验不同AI的特点

**任务：**
1. 用同一个问题，问2-3个不同的AI
2. 比较回答的差异
3. 分析：哪个AI的哪个方面表现更好？

**推荐问题：**
```
"未来10年，哪些职业最可能被AI替代？
作为一个年轻人，应该如何准备？"
```

---

### 练习2.4：质量校验实战（45分钟）

**目标：** 学会对AI输出负责

**任务：**
1. 让AI帮你分析一个你关心的决策（比如：要不要考研/跳槽/买房）
2. 用"校验清单"审核AI的回答
3. 标注你认为AI说错的/不适用的地方
4. 写出你最终的决定（不能只依赖AI）

## 模块三练习：AI批判与伦理

【开放练习，无标准答案】

### 练习3.1：偏见审计（60分钟）

**目标：** 学会识别AI中的偏见

**任务：**
1. 选择一个你常用的AI产品/功能
2. 用不同方式提问，观察AI回答的差异

**示例测试方向：**
```
• 用不同的名字问相同的问题（测试种族/性别偏见）
• 用不同地区背景问相同问题（测试地域偏见）
• 问涉及"传统"vs"现代"价值观的问题
```

3. 记录你发现的任何偏见
4. 思考：这个偏见的影响是什么？

---

### 练习3.2：权力分析（45分钟）

**目标：** 理解AI背后的权力结构

**任务：** 选择一个AI产品，完成以下分析：

```
产品名称：________________

1. 这个产品谁做的？（公司背景）
2. 他们的商业模式是什么？（怎么赚钱）
3. 你使用这个产品，他们获得什么？
4. 你贡献的数据，他们怎么用？
5. 他们的利益和你的利益一致吗？
6. 如果这个公司倒闭了，会影响你什么？
```

---

### 练习3.3：信息溯源（60分钟）

**目标：** 培养对AI生成内容的批判性思维

**任务：**
1. 让AI写一篇关于"XX问题"的综述
2. 标注AI提到的每个"事实"
3. 用其他来源核实至少5个"事实"
4. 统计：AI说的事实，有多少是准确的？

## 模块四练习：AI时代个人发展

【开放练习，无标准答案】

### 练习4.1：AI学习法实验（持续1周）

**目标：** 建立自己的AI辅助学习流程

**任务：**
1. 选择一个你想学习的具体主题
2. 设计一个为期1周的AI辅助学习计划
3. 每天使用AI学习，记录：
   - 今天用了AI做什么？
   - AI帮了你什么？
   - 你自己的思考和理解是什么？
4. 周末总结：AI辅助学习的优缺点

---

### 练习4.2：创作放大（45分钟）

**目标：** 体验AI辅助创作的正确方式

**任务：**
1. 构思一个你想要表达的观点/故事/想法
2. 用AI做以下步骤（但核心内容是你自己的）：
   - AI帮你扩展思路
   - AI帮你写初稿
   - 你改写核心部分
   - AI帮你润色
3. 最终作品：标注哪些是你的，哪些是AI辅助的
4. 反思：AI在创作中扮演了什么角色？

---

### 练习4.3：个人工作流设计（90分钟）

**目标：** 建立你的AI工作流

**任务：**
1. 盘点你日常生活/工作中的高频任务（至少10个）
2. 评估每个任务：是否可以用AI？AI适合做什么部分？
3. 选择2个最重要的场景，设计完整的人机协作流程
4. 实际运行一周，记录效果
5. 迭代优化

---

## 模块五练习：前沿与未来

【开放练习，无标准答案】

### 练习5.1：Agent体验（45分钟）

**目标：** 体验AI Agent的能力

**任务：**
1. 找一个支持Agent功能的AI产品（如有）
2. 尝试让Agent完成一个多步骤任务
3. 观察：Agent是如何分解任务的？它如何处理错误？
4. 思考：Agent的边界在哪里？

---

### 练习5.2：AGI辩论（60分钟）

**目标：** 形成对AGI的理性看法

**任务：**
1. 收集3个支持"AGI可能在10年内实现"的观点
2. 收集3个支持"AGI还很遥远或不可能"的观点
3. 收集1个你认为最有力的反驳
4. 写出你自己的立场，并说明理由

## 综合练习

【开放练习，无标准答案】

### 综合项目：设计你的人机协作方案（2-3小时）

**选择一个你真实面对的场景：**
- 准备一场重要面试
- 写一篇深度分析文章
- 做一个重大人生决定
- 规划一次复杂项目

**完成以下设计：**
```
1. 我的目标：________________

   人的工作：________________
   AI的工作：________________
3. 人机回环设计：
   第1轮：_______________
   第2轮：_______________
   第3轮：_______________
4. 质量校验方式：_______________
5. 我作为人类必须把控的环节：_______________
6. 风险点（AI可能出错的地方）：_______________
7. 最终验收标准：_______________
```
