## 推荐阅读

### 入门书籍

| 书名 | 作者 | 简介 |
|------|------|------|
| 《人工智能：一种现代方法》 | Stuart Russell / Peter Norvig | AI领域经典教材 |
| 《AI 2041》 | 李开复 / 陈楸帆 | 用故事预见AI未来 |
| 《人类简史》 | 尤瓦尔·赫拉利 | 思考人类与技术的关系 |
| 《思考，快与慢》 | Daniel Kahneman | 理解人类思维，识别人机差异 |
| 《原则》 | Ray Dalio | 决策思维，AI辅助决策参考 |

## 延伸学习路径

### 路径一：深度技术理解

```
入门 → CS50 AI → fast.ai → 斯坦福CS229
适合：想深入理解AI技术原理的人
```

### 路径二：批判性使用

```
本课程 → 提示工程课程 → 行业垂直应用
适合：想用AI提升工作效率的人
```

### 路径三：AI影响反思

```
本课程 → 《Life 3.0》→ 《The Alignment Problem》→ AI伦理课程
适合：关心AI社会影响的人
```

### 路径四：创造者路径

```
本课程 → Midjourney/SD → Runway → 独立创作
适合：想用AI进行内容创作的人
```
