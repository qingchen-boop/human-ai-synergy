> This document outlines the plan to translate the "人机共智：AI时代的协作思维与实践" (Human-AI Synergy Studies) course into English.

---

## Why Translate?

```
• Expand reach: Make the course accessible to English-speaking communities
• Global collaboration: Enable contributions from international learners
• Professional development: Useful for English-language education contexts
• Documentation: Many AI resources are primarily in English
```

## Translation Workflow

### Phase 1: Preparation

```
□ Set up parallel directory structure
  /en/ folder for English version

□ Create translation glossary
  Key terms mapping (Chinese → English)

□ Establish style guide
  - Tone: Semi-formal, accessible
  - Use "AI" as abbreviation (not "A.I.")
  - Preserve Chinese examples where culturally relevant
  - Add parenthetical translations for key concepts
```

### Phase 2: Core Translation (by priority)

```
Step 1: README + 学科纲要 (Lead with accessible content)
Step 2: Core modules (Module 1-5)
Step 3: Supporting materials (实践/评估/指南)
Step 4: Supplementary files
```

### Phase 3: Review & Refinement

```
□ Native English speaker review (if available)
□ Consistency check across all files
□ Terminology alignment
□ Cultural adaptation notes
```

### Handling Chinese-Specific Content

```
For examples that rely on Chinese context:
□ Adapt to international equivalents where possible
□ Keep original Chinese example in parentheses if culturally specific
□ Add context notes for non-Chinese readers

Example:
"高考" → "college entrance examination (Gaokao, China's national exam)"
"内卷" → "involution" (with brief explanation)
```

## Contribution Guidelines


```
□ Faithful to original meaning (not literal word-for-word)
□ Natural English expression
□ Consistent terminology
□ Preserve formatting (headers, lists, tables)
□ Keep code blocks as-is
```
