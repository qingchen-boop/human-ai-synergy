# 人机共智：AI时代的协作思维与实践 - 课程设计指南

---

## 课程基本信息

**课程名称：** 人机共智：AI时代的协作思维与实践 (Human-AI Synergy Studies)  
**课程代码：** HASS-101  
**建议学时：** 32学时（16周 × 2学时）
**建议年级：** 大学本科任意年级 / 研究生 / 成人学习者  
**班级规模：** 20-30人（小班更佳，便于讨论）

---

## 课程大纲

### 总体安排

| 周次 | 模块 | 主要内容 |
|------|------|----------|
| 1 | 导论 | 课程介绍、AI与人机协作的时代背景 |
| 2-3 | 模块一 | AI能力与局限 |
| 4-5 | 模块二 | 人机协作策略 |
| 6-7 | 模块三 | AI批判与伦理 |
| 8 | **期中** | 阶段回顾 + 个人展示 |
| 9-10 | 模块四 | AI时代个人发展 |
| 11-12 | 模块五 | 前沿与未来 |
| 13-14 | 综合 | 人机协作方案设计 |
| 15 | 综合 | 方案展示与互评 |
| 16 | 总结 | 课程回顾与展望 |

## 教学设计

### 第一周：导论与课程介绍

**学时：** 2学时

**教学目标：**
- 理解课程的核心理念和框架
- 认识到AI时代人机协作的必要性
- 建立学习社群的初步信任

**活动设计：**
```
课前（学生完成）：
- 阅读课程大纲
- 自我介绍：你和AI的关系

课堂（90分钟）：
1. 破冰讨论（20分钟）：你和AI打过交道吗？什么体验？
2. 课程介绍（20分钟）：核心理念、五模块框架
3. 小组活动（30分钟）：讨论"人机协作 vs 人vs机"
4. 个人反思（20分钟）：这门课你想学到什么？
```

---

### 模块一教学：AI能力与局限（第2-3周）

**学时：** 4学时

**教学目标：**
- 理解大模型基本原理
- 识别AI的能力边界
- 建立健康的AI信任度

**周次1（第2周）：AI是什么**

```
课前：
- 阅读：模块一内容
- 体验：用AI问5个问题，记录你的感受

课堂（2学时）：
1. 分享体验（20分钟）：你问AI了什么？感觉如何？
2. 讲解：LLM工作原理（直观类比）
3. 活动：AI能力边界测试（用提前设计的问题测试AI）
4. 讨论：什么让你惊讶？什么和你想的不一样？
```

**周次2（第3周）：AI的局限**

```
课前：
- 阅读：模块一剩余内容
- 任务：让AI介绍你专业领域的一个概念，记录是否准确

课堂（2学时）：
1. 幻觉案例分享（20分钟）：你的测试结果如何？
2. 讲解：三大局限
3. 活动：设计你自己的"AI适合度测试"
4. 练习：任务分类练习
```

---

### 模块二教学：人机协作策略（第4-5周）

**学时：** 4学时

**周次3（第4周）：提示工程基础**

```
课前：
- 阅读：模块二前两节
- 练习：优化3个你自己的"不好提示"

课堂（2学时）：
1. 展示：你优化的提示，解释为什么更好
2. 讲解：提示四要素
3. 练习：现场设计提示（小组竞赛）
4. 实践：实际运行你设计的提示
```

**周次4（第5周）：任务分解与迭代**

```
课前：
- 阅读：模块二后两节
- 任务：用"分解法"完成一个小任务，记录过程

课堂（2学时）：
1. 过程分享（20分钟）：分解策略的效果
2. 讲解：任务分解与人机分配
3. 讲解：迭代回环
4. 练习：设计一个需要多次迭代的任务
```

---

### 模块三教学：AI批判与伦理（第6-7周）

**学时：** 4学时

**周次5（第6周）：数据偏见与权力**

```
课前：
- 阅读：模块三前两节
- 任务：选择身边一个AI产品，分析其利益关系

课堂（2学时）：
1. 产品分析展示（20分钟）：你分析的产品
2. 讲解：数据偏见
3. 活动：偏见测试（用不同方式提问，观察差异）
4. 讲解：权力结构
```

**周次6（第7周）：就业与知识生产**

```
课前：
- 阅读：模块三后两节
- 思考：你所在的行业，AI可能带来什么影响？

课堂（2学时）：
1. 小组讨论（30分钟）：AI对你行业的影响
2. 辩论（40分钟）：AI是否会让大量人失业？
3. 个人反思（20分钟）：你能做什么准备？
```

---

### 期中（第8周）

**学时：** 2学时

```
课前：
- 完成：一份个人AI工作流设计（草稿）
- 准备：3分钟展示

课堂（2学时）：
1. 小组展示（60分钟）：每人3分钟，其他同学提问
2. 互评（20分钟）：给同学反馈
3. 老师总结（10分钟）：共性问题和改进方向
```

---

### 模块四教学：AI时代个人发展（第9-10周）

**学时：** 4学时

**周次7（第9周）：AI辅助学习与创作**

```
课前：
- 阅读：模块四前两节
- 任务：用AI辅助学习一个概念，提交过程记录

课堂（2学时）：
1. 学习经验分享（30分钟）
2. 讲解：AI辅助学习的正确打开方式
3. 实践：设计一个AI辅助学习方案
```

**周次8（第10周）：个人工作流设计**

```
课前：
- 阅读：模块四后两节
- 任务：开始运行你的个人AI工作流

课堂（2学时）：
1. 运行分享（30分钟）：遇到什么问题？
2. 迭代优化讨论（30分钟）
3. 边界讨论（30分钟）：什么时候不该用AI？
```

### 模块五教学：前沿与未来（第11-12周）

**学时：** 4学时

**周次9（第11周）：Agent与多模态**


## 教师指南

### 常见问题处理

| 场景 | 建议 |
|------|------|
| 学生过度依赖AI | 强调"你必须为AI的输出负责" |
| 学生完全不用AI | 鼓励尝试，但不强求 |
| 学生觉得AI无所不能 | 用案例展示AI的失败和局限 |
| 学生对AI过于恐惧 | 用实践建立理性认知 |
| 讨论偏离主题 | 拉回具体场景和案例 |

### 评估建议

- 过程评估比结果更重要
- 鼓励学生真实反思，不要表演"正确答案"
- 关注学生是否形成了自己的判断
