## 第一课：AI Agent——从工具到行动者

### 什么是Agent？

**简单理解：** 不只是回答问题，而是能**替你做事**的AI。

```
传统AI：你说"帮我订明天去北京的机票" → AI给你链接/建议
Agent：  你说"帮我订明天去北京的机票" → AI直接帮你订完
```

### Agent的能力层级

| 层级 | 能力 | 例子 |
|------|------|------|
| L1 | 响应查询 | 回答问题 |
| L2 | 提供建议 | 推荐方案 |
| L3 | 自主行动 | 替你执行（需要授权） |
| L4 | 长期规划 | 自主设定目标并完成 |
| L5 | 跨系统协作 | 在多个系统间协调工作 |

### 当前Agent的发展状态

**已经能做到：**
- 预约、订票、发送邮件
- 浏览网页、填写表单
- 调用多个工具完成复杂任务
- 简单的多步骤自主规划

**还在发展中：**
- 长期记忆（每次对话从零开始）
- 精确的时间管理
- 复杂任务的可靠执行
- 对不确定情况的正确判断

### Agent带来的新问题

```
1. 责任归属：如果Agent替你做的事情出了问题，谁负责？
2. 安全边界：Agent能自主做什么？需要哪些审批？
3. 授权管理：Agent能用你的身份做什么？
4. 失控风险：自主性越高，出错后果越严重
```

## 第二课：多模态AI

### 什么是多模态？

**单模态：** 只处理一种类型（文字/图片/声音）  
**多模态：** 同时处理多种类型，理解它们之间的关系

```
多模态AI可以：
• 看一张图片，写出描述
• 听一段话，转录并翻译
• 看视频，理解发生了什么
• 结合文字和图片，产生新的理解
```

### 当前能力

| 能力 | 状态 | 说明 |
|------|------|------|
| 图像理解 | 成熟 | 能识别、描述、解释图片 |
| 视频理解 | 发展中 | 能理解短视频内容 |
| 语音交互 | 成熟 | 实时对话、翻译 |
| 图像生成 | 成熟 | 但细节可能有错误 |
| 视频生成 | 发展中 | 短片质量已不错 |

### 多模态带来的影响

**积极面：**
- 更自然的人机交互
- 盲人/聋哑人可以获得更强的辅助
- 内容创作门槛进一步降低

**担忧面：**
- 深度伪造（换脸、声音克隆）更难辨别
- 虚假信息的载体从文字扩展到视频
- AI生成内容的边界更加模糊

## 第三课：我们离AGI还有多远？

### 什么是AGI？

**通用人工智能（AGI）**：能够完成任何人类能完成的智力任务的AI。

### 当前观点光谱

| 立场 | 代表人物/群体 | 预测 |
|------|--------------|------|
| 乐观派 | 一些AI研究者 | 5-20年内可能实现 |
| 谨慎派 | 大多数严肃AI学者 | 不确定，可能需要几十年 |
| 怀疑派 | 部分认知科学家 | AGI可能根本不是正确的目标 |
| 末日派 | 一些AI安全研究者 | AGI可能危险，需要提前布局 |

### 为什么很难预测？

```
1. 我们不知道"智能"到底是什么
2. 我们不知道人脑的全部机制
3. 我们对"意识"和"理解"没有共识
4. 历史告诉我们预测技术发展总是错的
5. 可能有我们还没想到的瓶颈
```

### 理性态度

```
✅ 保持开放：既不排除AGI可能，也不过度乐观
✅ 关注当下：更重要的是今天AI的能力和局限
✅ 做好准备：无论AGI何时到来，提升人机协作能力都有价值
✅ 参与讨论：AGI的走向是社会问题，不是纯粹的技术问题
```

## 第四课：人机关系的哲学思考

### 我们在创造什么？

**几种可能的未来图景：**

#### 1. 工具论
> AI是更高级的工具，人类保持控制

**优点：** 符合现有认知框架  
**问题：** 如果AI越来越自主，"工具"的定义还成立吗？

#### 2. 伙伴论
> 人类和AI是平等的协作者

**优点：** 尊重AI的能力  
**问题：** AI没有利益诉求，"平等"有意义吗？

#### 3. 进化论
> 人类和AI将融合，产生新的存在形式

**优点：** 超越二元对立  
**问题：** 这还是"人类"吗？是我们想要的吗？

#### 4. 生态论
> 人机关系像自然生态系统，没有单一控制者

**优点：** 避免权力集中  
**问题：** 如何在这种复杂系统中保持人类福祉？

### 我们应该问的问题

```
□ 我们想让AI成为什么？（工具、伙伴、还是别的？）
□ 我们愿意为AI让渡多少自主权？
□ 如果AI有"利益"，谁来代表它们？
□ 什么应该由AI决定？什么必须由人决定？
□ 我们希望我们的孩子在一个什么样的AI世界中成长？
```

## 第五课：AI治理的未来方向

### 什么是AI治理？

**AI治理 = 社会如何决定AI能做什么、应该做什么**

### 当前主要议题

| 议题 | 核心问题 |
|------|----------|
| 透明度 | AI决策过程是否应该公开？ |
| 责任 | AI造成的损害谁负责？ |
| 隐私 | 个人数据如何保护？ |
| 安全 | 如何防止AI被滥用？ |
| 公平 | 如何防止AI歧视？ |
| 控制 | 人类如何保持对AI的控制？ |

### 主要治理路径

```
路径A：法规主导（欧盟AI法案）
→ 政府通过法律规范AI的使用

路径B：行业自律（现有多数情况）
→ AI公司自我约束，但缺乏强制力

路径C：技术解决（-alignment研究）
→ 通过技术让AI本身更安全

路径D：公众参与（民主决策）
→ 公民社会共同讨论AI的边界
```

### 公民能做什么？

```
□ 了解你使用的AI产品的政策和限制
□ 支持AI伦理和公平性的社会倡导
□ 参与公共讨论，不把AI问题全交给"专家"
□ 在自己能力范围内，促进AI的负责任使用
□ 培养下一代对AI的批判性思维
```
