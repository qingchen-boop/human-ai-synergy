# 模块二补充：提示工程模板库

---

## 模板使用说明

以下模板可以直接复制使用。**[]** 中的内容需要替换成你的具体情况。

---

## 一、通用基础模板

### 1.1 标准问答模板

```
角色：你是一个[领域]专家。
背景：我正在[具体场景/目的]。
任务：请[具体要求]。
约束：
- [约束条件1]
- [约束条件2]
格式：请以[格式]输出。
```

**示例：**
```
角色：你是一个资深科技记者。
背景：我需要写一篇关于AI对教育影响的报道。
任务：请分析AI在K-12教育中的应用现状和挑战。
约束：
- 面向普通读者，避免过度专业术语
- 需要包含至少3个真实案例
- 指出潜在的风险和争议
格式：请以新闻报道风格，分3个部分，每个部分300字左右。
```

---

### 1.2 苏格拉底式提问模板

```
我想理解[主题/概念]。

我目前的理解是：[写下你的理解]

请用苏格拉底式提问：
1. 你为什么这么认为？
2. 这个观点有没有反面例子？
3. 如果这个理解是错的，最可能出错在哪里？
4. 这个观点在什么情况下可能不适用？
```

---

### 1.3 自我反思模板

```
我刚刚阅读/学习的内容是：[简要描述]

请用你自己的话解释这个概念：
[写下你的理解]

如果有不清楚的地方，请指出：
1. __________
2. __________
3. __________

请针对这些不清楚的地方，给我一个更简单的解释。
```

## 二、学习辅助模板

### 2.1 概念解释模板

```
请用[难度级别：小学/初中/高中/大学/专家]的水平，
解释"[概念名称]"。

要求：
- 使用至少2个生活化的例子
- 避免使用专业术语（如果必须使用，请附上解释）
- 帮我建立一个直观的理解，而不是抽象的定义
```

---

### 2.2 知识比较模板

```
请比较[概念A]和[概念B]。

维度：
1. 定义/本质
2. 主要特征
3. 应用场景
4. 优点
5. 缺点/局限性
6. 它们有什么关联？

最后，请用一个比喻来总结它们的区别。
```

---

### 2.3 学习测试模板

```
我刚学习了[主题]。请问我几个关键问题来测试我的理解。

要求：
- 3-5个问题
- 包括记忆题（事实）、理解题（概念）、应用题（场景）
- 不要问太细节的刁钻问题
- 最后请评估我的回答，并指出我理解不到位的地方。
```

---

### 2.4 知识总结模板

```
我刚读完/学完[内容来源：书名/课程/文章]。

请帮我：
1. 提取3-5个核心观点
2. 用一句话总结每个核心观点
3. 找出与[相关领域/你已知的知识]的联系
4. 提出2-3个我应该继续追问的问题
```

## 三、写作辅助模板

### 3.1 邮件写作模板

```
场景：我需要写一封[类型]邮件给[收件人身份]。

邮件目的：[你想达到的目标]
关键信息：
- [信息1]
- [信息2]
- [信息3]

语气：[正式/半正式/友好/恳求]
长度：[简短/中等/详细]
特殊要求：[如果有]

请帮我写这封邮件。
```

---

### 3.2 报告/文章框架模板

```
请为"[主题]"写一个文章/报告框架。

背景：
- 受众：[谁会读这篇文章]
- 目的：[你想让读者获得什么/做什么]
- 调性：[专业/轻松/学术/大众]

要求：
- 包含引言、正文（至少3个部分）、结论
- 每个部分需要包含的内容要点
- 建议的段落长度
- 至少2个小案例或数据支撑
```

---

### 3.3 头脑风暴模板

```
我想为[场景/问题]想一些创意解决方案。

约束条件：
- [约束1]
- [约束2]
- 至少需要[数量]个想法

请提供多样化的想法，包括：
1. 保守可行的方案
2. 激进创新的方案
3. 意想不到的跨界方案

每个想法请说明：
- 核心思路
- 优点
- 可能的风险/挑战
```

---

### 3.4 演讲/PPT大纲模板

```
请为"[演讲主题]"设计一个演讲大纲。

场景：
- 时长：[X分钟]
- 受众：[受众背景]
- 目的：[说服/告知/激励]

要求：
- 开场方式：请提供一个有吸引力的开场
- 至少3个主要论点
- 每个论点需要的数据/案例支持
- 结束语：呼吁行动/金句收尾
- 建议的过渡句
```

---

### 3.5 简历优化模板

```
我正在申请[职位/公司]。

我的背景：
- 学历：[简要]
- 经验：[简要，列出2-3个相关经历]
- 技能：[列出与申请职位相关的技能]

请帮我：
1. 突出我与这个职位最相关的经历
2. 用更专业的语言描述我的工作
3. 量化我的成就（加入具体数字）
4. 指出我还缺少什么，如何弥补
```

## 四、决策辅助模板

### 4.1 利弊分析模板

```
我正在考虑[决定内容]。

我的情况：
- [背景信息1]
- [背景信息2]
- 我的顾虑：[列出你的担忧]

请帮我：
1. 分析这个决定的可能利弊
2. 站在不同立场（支持/反对/中立）分析
3. 指出我可能忽略的因素
4. 问我一些能帮助决策的关键问题
5. 最后不要替我做决定，但可以给出建议框架
```

---

### 4.2 风险评估模板

```
我想评估"[行动/决定]"的风险。

请分析：
1. 最坏的情况是什么？
2. 发生的可能性有多大？（高/中/低）
3. 如果发生，影响有多大？
4. 我可以如何准备/应对？
5. 这个风险值得我承担吗？

我的具体情况：[补充背景]
```

---

### 4.3 视角扩展模板

```
我倾向于[你的观点/选择]。

但我想听听不同意见。

请从以下角度挑战我的观点：
1. 实用角度：现实中最可能出什么问题？
2. 道德角度：这个选择有什么伦理问题？
3. 长期角度：5年后我会怎么看这个决定？
4. 对方视角：如果我是受影响最大的人，我怎么想？

请给出具体的反驳，而不是泛泛而谈。
```

## 五、代码/技术模板

### 5.1 代码解释模板

```
请解释以下代码做了什么：

[粘贴代码]

要求：
- 逐行或按块解释
- 用简单的话解释每个部分

- 如果有更优的实现方式，请建议
```

- 已有技能：[你已会的相关技能]

3. 推荐3个最适合我的学习资源

## 六、创意创作模板

### 6.1 故事创作模板

```
请帮我创作一个关于[主题]的[短篇故事/场景]。

要求：
- 主角：[简单描述]
- 冲突/核心矛盾：[故事的核心问题]
- 调性：[悬疑/温馨/搞笑/励志/暗黑]
- 字数/长度：[X字/3分钟阅读]
- 至少包含一个出人意料的转折

我的偏好：
- [偏好1]
- [偏好2]
```

---

### 6.2 角色扮演模板

```
请扮演"[角色]"。

背景：我是"[你的角色]"，我需要和你进行"[场景]"的练习。

我的目标：[你想从这次练习中获得什么]

规则：
- 请保持角色
- 适时挑战我、提问我
- 结束时给我反馈

让我们开始。
```

**示例场景：**
- 面试官-求职者
- 甲方-乙方
- 销售员-顾客
- 心理咨询师-来访者

## 八、高级模板

### 8.1 思维链模板

```
问题：[你的问题]

请用以下步骤回答：
1. 首先，分解这个问题，找出核心要素
2. 其次，针对每个要素进行分析
3. 然后，找出要素之间的关联
4. 最后，综合所有分析，给出答案

在每一步，请说明你的推理过程。
```

---

### 8.2 多AI协作模板

```
我需要完成一个复杂的任务：[任务描述]

我计划这样分工：
1. AI-A（[选择：擅长逻辑/创意/分析]的AI）：[任务1]
2. AI-B（[选择]的AI）：[任务2]
3. 我自己：[任务3]

请帮我设计一个更详细的工作流，
包括：
- 每一步的具体任务
- 如何在AI之间传递信息
- 如何校验AI的输出
```

---

### 8.3 自我改进模板

```
我刚刚完成了一个任务：[描述任务]

我使用了以下方法：[描述你用的方法]

结果：[描述结果]

请帮我分析：
1. 什么做得好？（可以复用）
2. 什么可以改进？
3. 下次我可以尝试的改进方案是什么？

请给我具体的、可执行的建议。
```

## 模板使用技巧

### 1. 不要照搬，调整才有灵魂
模板是起点，不是终点。根据你的实际情况调整措辞和结构。

### 2. 迭代比完美更重要
第一次用模板可能效果不完美，多用几次，自然知道哪里该调整。

### 3. 组合使用
一个复杂的任务可以同时使用多个模板——比如先用头脑风暴，再用写作框架。

### 4. 记录你的成功模板
用得好的模板，记下来，形成自己的"提示词库"。
