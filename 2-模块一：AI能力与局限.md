# 模块一：AI能力与局限
## Understanding AI

---

## 学习目标

- 理解大语言模型的基本工作原理（直觉级理解）
- 识别AI的能力边界，知道什么任务适合AI
- 理解AI"幻觉"产生的原因，并掌握应对策略
- 建立对AI输出的健康信任度

---

## 第一课：大模型是怎么工作的？

### 直观类比：超级 autocomplete

想象你有一本读完了人类所有文字的书。然后给你一个词，它能猜到下一个最可能出现的词是什么——不是靠"理解意思"，而是靠"见过太多例子，概率最高"。

**这就是大语言模型（LLM）的本质：**

```
输入 → 预测下一个词（概率分布）→ 输出
         ↑ 你给的压力（prompt）
```

### 三个关键认知

1. **AI不"理解"** —— 它做的是模式匹配和概率预测，不是真正的理解
2. **AI没有"知识"** —— 它有的是文字规律，知识是副作用
3. **AI没有"观点"** —— 它的输出是统计结果，不是思考结论

### 案例分析

| 人类说 | AI实际做的 |
|--------|-----------|
| "解释量子力学" | 找到一个最可能"解释量子力学"的文字序列 |
| "你觉得AI危险吗" | 生成一个看起来像"回答AI危险问题"的句子 |

## 第二课：AI的三大能力

### 1. 文本生成（Generation）

**强项：**
- 写作初稿、头脑风暴
- 格式转换（总结、扩写、改写）
- 模拟某种风格或角色

**弱项：**
- 产生真正全新的想法（它只能重组已知模式）
- 知道你不知道什么（它不知道自己的知识边界）

### 2. 模式识别（Pattern Recognition）

**强项：**
- 分类、归类
- 找规律
- 识别异常

**弱项：**
- 在噪声大的数据中可能误判
- 对抗性样本（刻意构造的误导输入）

### 3. 推理与计算（Reasoning）

**强项：**
- 逻辑推理（当规则清晰时）
- 数学计算（基础运算）
- 多步骤规划（大模型有"思维链"能力）

**弱项：**
- 复杂数学（超出训练分布的高难度问题）
- 实时信息（不知道最新发生的事）
- 精确事实记忆（容易张冠李戴）

## 第三课：AI的三大局限

### 局限一：幻觉（Hallucination）

**什么是幻觉：** AI一本正经地胡说八道，看起来非常可信但实际是错的。

**为什么产生：**
1. 训练数据包含错误信息 → AI学会了错误模式
2. 概率预测机制 → 总是要输出点什么，哪怕不确定
3. 没有"我不知道"的约束 → 倾向于编造答案来维持对话流畅

**幻觉案例：**
```
用户：介绍一下2024年诺贝尔物理学奖得主
AI：[生成一个听起来很真实但名字错误的人]
```

**应对策略：**

| 策略 | 操作 | 适用场景 |
|------|------|----------|
| 交叉验证 | 用另一个AI或搜索引擎核实 | 重要事实 |
| 要求引用 | 让AI给出信息来源 | 研究类任务 |
| 反向提问 | "你怎么知道这个是准确的？" | 判断AI自信度 |
| 拆分提问 | 每次只问一个具体事实 | 复杂问题 |

### 局限二：边界模糊（Boundary Blur）

AI不知道自己**不知道**什么。

```
知道 + 不知道 → 混淆在一起，都用同样的自信语气输出
```

**后果：**
- 用户无法从语气判断AI是否真的懂
- AI可能在不擅长的领域也给出流畅但错误的答案

**应对策略：**
- 养成"不确定就查"的习惯
- 重要决策不依赖单一AI来源

### 局限三：时效性（Recency）

**问题：** 训练数据有截止日期，AI不知道截止日期之后发生的事。

**举例：**
- 不知道最新的新闻
- 不知道刚发布的法律/政策
- 不知道新出的软件版本变化

**应对策略：**
- 了解你使用的AI的训练数据截止日期
- 时效性强的任务用实时搜索补充

## 第四课：任务分类练习

**核心能力：判断一个任务适不适合交给AI**

### 高AI适配度任务

| 任务类型 | 原因 | 示例 |
|----------|------|------|
| 初稿生成 | 需要大量文本，AI效率高 | 写邮件、写报告框架 |
| 头脑风暴 | 需要发散思维，AI擅长组合 | 活动创意、名字建议 |
| 格式转换 | 规则清晰，机械操作 | 总结、翻译、改写 |
| 解释概念 | 有标准答案的知识类问题 | 解释某个理论 |
| 角色扮演 | 模拟某种风格或视角 | 面试练习、谈判模拟 |

### 低AI适配度任务

| 任务类型 | 原因 | 示例 |
|----------|------|------|
| 最新事实查询 | 时效性强，AI可能错误 | 今天股市收盘价 |
| 个人化决策 | 需要你的独特情况 | 职业选择、买房建议 |
| 主观判断 | 需要你的价值观 | 评价一个作品 |
| 高风险任务 | 错误后果严重 | 医疗建议、法律咨询 |
| 创意验证 | 需要真实世界反馈 | 这个商业点子真的可行吗 |

### 练习：判断以下任务适合用AI吗？

1. 帮我写一封请假邮件
2. 帮我查一下2026年春节是几号
3. 告诉我该不该辞职
4. 帮我分析这篇论文的主要贡献
5. 帮我决定买哪只股票

## 第五课：建立健康信任度

### 信任度光谱

```
低置信度 ←————————————————————————→ 高置信度
   ↑                                    ↑
简单事实查询(需核实)              格式转换/风格模仿
   ↑                                    ↑
创意发散                      有明确正确答案的知识问题
```

### 训练方法：每收到AI回答，先问

1. **这个回答涉及的事实，有办法核实吗？**
2. **这个问题有没有可能是AI不擅长的领域？**
3. **AI的语气是自信还是犹豫的？**（自信不代表准确）
